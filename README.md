A mod loader for Subnautica 2 with build in mod manager, installer and option to edit mod config.

Installation instructions
1. Extract the Zip to a dedicated folder anywhere on your computer.
2. Open  sub loader2.exe

Main features
- Mod manager,
- Turn on/off mods,
- Edit config of the mods installed,
- Install mods.
  
Requirements
UE4SS
